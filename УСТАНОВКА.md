# Как подключить Huntflow к Claude — простая инструкция

Эта инструкция для тех, кто **не программист**. После настройки вы сможете прямо в Claude
писать запросы вроде «покажи кандидатов на вакансию» или «собери аналитику по вакансии»,
и Claude возьмёт данные из вашего Huntflow. Всё займёт **10–15 минут**, делается один раз.

---

## Что нужно
- Компьютер (Mac или Windows).
- Приложение **Claude Desktop**.
- Доступ к вашему аккаунту **Huntflow** (чтобы получить токен).

---

## Шаг 1. Установите Claude Desktop
1. Откройте сайт **claude.ai/download**
2. Скачайте версию для вашего компьютера и установите (как обычную программу).
3. Запустите и войдите в свой аккаунт Claude.

## Шаг 2. Установите Node.js (это «движок», нужен один раз)
1. Откройте сайт **nodejs.org**
2. Скачайте кнопку **LTS** (рекомендуемая версия).
3. Установите, нажимая «Далее/Continue» до конца. Ничего настраивать не надо.

## Шаг 3. Получите токен Huntflow
Токен — это ключ доступа. Без него Claude не сможет читать ваши вакансии.
1. Зайдите в **Huntflow → Настройки → API** (раздел токенов).
2. Нажмите **«Добавить токен»**, придумайте название → **Создать**.
3. Появится **ссылка** вида `https://huntflow.ai/token_request/...`
4. **Откройте эту ссылку в браузере** (вы должны быть авторизованы в Huntflow) и нажмите **«Получить» (Receive)**.
   > ⚠️ Не пересылайте ссылку и не открывайте её «просто посмотреть» заранее — после открытия она срабатывает один раз.
5. Скопируйте появившуюся **длинную строку из букв и цифр** — это ваш токен. **Никому его не показывайте.**

## Шаг 4. Впишите Huntflow в настройки Claude
1. В Claude Desktop откройте **Настройки** (Settings).
2. Перейдите в раздел **Developer** (Разработчик) → нажмите **Edit Config** (Редактировать конфигурацию). Откроется текстовый файл.
3. Вставьте в него такой блок (если файл пустой — вставьте целиком):

```json
{
  "mcpServers": {
    "huntflow": {
      "command": "npx",
      "args": ["-y", "@gaivoronsky/huntflow-mcp"],
      "env": {
        "HUNTFLOW_TOKEN": "сюда-вставьте-ваш-токен",
        "HUNTFLOW_BASE_URL": "https://api.huntflow.ai/v2"
      }
    }
  }
}
```

4. Замените `сюда-вставьте-ваш-токен` на токен из Шага 3 (вставьте между кавычками).
5. **Сохраните** файл (Cmd+S на Mac / Ctrl+S на Windows).

> Если в файле уже есть другие сервисы — просто добавьте строку `"huntflow": { ... }`
> внутрь существующего `"mcpServers"`, не удаляя остальное.

## Шаг 5. Перезапустите Claude
Полностью закройте Claude Desktop (Mac: ⌘Q; Windows: выход из приложения) и откройте снова.
Это обязательно — иначе изменения не подхватятся.

## Шаг 6. Проверьте, что работает
Напишите Claude, например:
- «Покажи список аккаунтов в Huntflow»
- «Покажи открытые вакансии»

Если Claude отвечает данными из Huntflow — всё готово! 🎉

---

## Что теперь умеет Claude
- 📋 Вакансии: список, детали, сколько дней в работе.
- 👤 Кандидаты: поиск, карточки, резюме, на каких этапах находятся.
- 🔄 Воронка: этапы подбора, причины отказа.
- 💬 Комментарии о кандидатах: читать и добавлять.
- 📊 **Готовые сценарии** (в Claude они называются «промпты»): аналитика по вакансии —
  дни в работе, кандидаты у заказчика, сроки отправки CV клиенту.

---

## Если что-то не работает
- **Claude не видит Huntflow** → проверьте, что полностью перезапустили приложение (Шаг 5).
- **Ошибка про токен / 401** → токен мог истечь (живёт ~7 дней). Получите новый по Шагу 3 и впишите заново.
- **Пустые ответы / ошибки 404** → если ваш аккаунт на российском Huntflow, замените в настройках
  `https://api.huntflow.ai/v2` на `https://api.huntflow.ru/v2`.
- **Совсем не запускается** → убедитесь, что установили Node.js (Шаг 2) и сохранили файл настроек.

Если не получается — покажите этот файл и текст ошибки тому, кто настраивал, он быстро поможет.
